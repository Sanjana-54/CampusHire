# 🚀 CampusHire — Smart Placement Tracker

CampusHire is a full-stack placement management platform designed to streamline campus recruitment activities for students and placement administrators.

The system enables students to discover eligible placement drives, track application progress, receive notifications, and manage their profiles. Placement administrators can manage recruitment drives, monitor applications, update recruitment statuses, and analyze placement statistics through a centralized dashboard.

---

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Bcrypt.js

---

## 🔄 System Overview

### 🎓 Student Portal

* View placement drives based on eligibility criteria.
* Apply for recruitment drives.
* Track application progress and recruitment status.
* Receive placement-related notifications.
* Manage personal profile and resume information.

### 💼 Admin Portal

* Create and manage placement drives.
* View and manage registered students.
* Monitor applications across companies.
* Update recruitment rounds and application statuses.
* Access placement statistics and analytics.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone <repository-url>
cd CampusHire
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
